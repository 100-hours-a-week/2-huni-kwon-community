# 2-huni-kwon-community
# 📌 프로젝트 개요

기본적인 게시판 기능을 갖춘 웹 애플리케이션입니다.

---

## 🚀 주요 기능

- 🔐 **회원 기능**
  - 로그인 / 회원가입
  - 회원정보 수정
  - 비밀번호 수정
  - 로그아웃

- 📝 **게시글 기능**
  - 게시글 작성
  - 게시글 목록 조회
  - 게시글 상세 조회
  - 게시글 수정 및 삭제
  - 좋아요

- 💬 **댓글 기능**
  - 댓글 작성
  - 댓글 수정 및 삭제

---

## 🛠️ 기술 스택

- **HTML**
- **CSS**
